units

notation

common abbreviations are (g)ram, (t)easpoon, (T)ablespoon, (c)up. units are written as variable factors: 3/4 cup = 3c/4; 3 tablespoons = 3T. a coefficient of 1 is omitted, so 48t = 16T = c = 1 cup.

conversions

metric / imperial (approximate)

  • c = 240ml
  • g = 28.5oz
  • lb = 456g

weight / volume (approximate, not valid far from sea level)

  • g water/water-based liquid = ml, so c = 240g
  • g oil/fat = 8ml/7, so c = 210g
  • c flour/milled grain = 144g
